# guarantorLens_mission_capstone_ML

ML repository for the ALU mission capstone project, including model training notebooks, datasets, results, visualizations, and model comparisons.

## Repository structure

```text
notebooks/            # Jupyter notebooks for exploration and model training
datasets/
  raw/                # Original input datasets
  processed/          # Cleaned/feature-engineered datasets
results/
  models/             # Trained model artifacts
  metrics/            # Evaluation outputs and metric summaries
  figures/            # Visualization images and plots
comparisons/          # Model-to-model comparison outputs
```
